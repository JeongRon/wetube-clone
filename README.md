## URL Plan

- Global 라우터
  - / -> Home
  - /join -> Join
  - /login -> Login
  - /search -> Search
- User 라우터
  - /users/:id -> See User
  - /users/logout -> Log Out
  - /user/edit -> Edit My Profile
  - /user/delete -> Delete My Profile
- Video 라우터
  - /videos/:id -> Watch Video
  - /videos/:id/edit -> Edit Video
  - /videos/:id/delete -> Delete Video
  - /videos/upload -> Upload Video
