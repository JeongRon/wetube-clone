## URL Plan

- Global 라우터
  - / -> Home
  - /join -> Join
  - /login -> Login
  - /search -> Search
- User 라우터
  - /user/edit -> Edit user
  - /user/delete -> Delete user
- Video 라우터
  - /videos/watch -> Watch Video
  - /videos/edit -> Edit Video
  - /videos/delete -> Delete Video
  - /videos/comment -> Comment on a Video
  - /videos/comment/delete -> Delete A Comment of a Video
